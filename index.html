About Lossless Scaling

Lossless Scaling allows you to scale windowed games to full screen using state-of-the-art scaling algorithms, as well as use ML-based proprietary scaling and frame generation.

  • Frame Generation: LSFG
  • Scaling Options: LS1, AMD FSR, NVIDIA Image Scaling, Integer Scaling, Nearest Neighbor, xBR, Anime4K, Sharp Bilinear, Bicubic CAS
  • Enhance games without native support for frame generation and upscale both modern and pixel-art games.
